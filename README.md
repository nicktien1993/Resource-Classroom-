
# 🍎 資源班數學教室 - AI 自動化講義與練習卷工具

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Status: Beta](https://img.shields.io/badge/Status-Beta-blue.svg)

這是一個專為**國小資源班/特教老師**設計的數學教學輔助工具。透過 AI (Gemini API) 的力量，幫助老師在 30 秒內生成符合學生程度、具備注音與視覺化圖示的專業講義。

## ✨ 核心功能
- **三大出版社同步**：支援康軒、南一、翰林版課程目錄查詢。
- **微步化教學**：講義範例自動拆解步驟，符合特教「小步子」教學原則。
- **一鍵練習卷**：根據講義內容，自動生成對應的計算題與應用題。
- **無障礙排版**：
  - **注音系統**：台灣標準教科書風格注音。
  - **視覺化 SVG**：自動生成圖解，幫助學生理解抽象概念。
  - **數位手寫板**：直接在螢幕上運算，適合電子白板或 iPad 教學。
- **高品質匯出**：一鍵轉換成 A4 PDF，方便直接列印使用。

## 🚀 如何開始使用？
1. 前往我們的 [GitHub Pages 連結](https://nicktien1993.github.io/Resource-Classroom-/) (請在開啟 Pages 功能後替換此網址)。
2. 點擊畫面上的 **「🔑 啟動 AI 教學助手」**。
3. 選取您的 API Key 專案（目前為測試階段，建議老師先自行申請免費金鑰）。
4. 選擇學年度與單元，開始您的教學！
